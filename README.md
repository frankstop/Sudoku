# Sudoku

A static Sudoku web game built with plain HTML, CSS, and JavaScript.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Features

- Easy, Medium, Hard, and Expert puzzles
- Mouse, touch, and keyboard input
- Notes mode, hints, erase, reset, undo, and redo
- Timer and completion state
- Mistake checking with inline highlights
- Responsive layout for desktop and phone screens
